Tasks for lesson 19 (L19.2)
################################

Learning goals:
Repetition
Update repository history from remote
Update repository from remote


1. Check the status of your repository

git status
git diff
git diff master origin/master


2. Download the history from remote repository

git fetch
git status
git diff
git diff master origin/master

Once you see the following message, continue to point 3

   "Your branch is behind 'origin/master' by 1 commit, and can be fast-forwarded.
      (use "git pull" to update your local branch)"


3. Update your repository with changes from the remote repository (close this file first)

git pull
git status
git diff master origin/master


4. When you have finished point 3, open this file again for further instructions



==== UPDATED INSTRUCTIONS =======

Learning goals:
Update local and remote fork

5. The usual update first IN CLONE OF ORIGINAL REPOSITORY

git branch
git checkout master_sara
git pull

Switch to your branch

git checkout <Your_chapter_name>
git branch
git merge master_sara

6. Now go to your local fork clone again. Update your clone with latest changes from the ORIGINAL repository (upstream), i.e. not your fork.

git branch
git checkout master_sara
git pull upstream master_sara

7. Compare the different repositories using either diff difftool (lesson 3)

git diff master_sara upstream/master_sara
git diff master_sara origin/master_sara
git diff upstream/master_sara origin/master_sara

8. Now update your remote fork

git push origin/master_sara

9. Compare the differences between the different repositories again

git diff master_sara upstream/master_sara
git diff master_sara origin/master_sara
git diff upstream/master_sara origin/master_sara
